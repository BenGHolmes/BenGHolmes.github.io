Projects

Autonomous Rescue Robot with Computer Vision

Course project (June - Aug 2018)

During this past summer, I participated in the yearly Engineering Physics robot design course. The course is structured as a competition, with the goal being to build a fully autonomous robot that is able to complete the given challenge in just 13 weeks. This year, we had to rescue Ewoks (stuffed animals) from the Empire stronghold by navigating over gaps, differentiating between various infrared light signals, and detecting and capturing 5 stuffed Ewoks along the way.

From the beginning, our team decided that we wanted to try to implement a universal solution, that required no hardcoding or prior knowledge of the course. If successful, our robot would be able to navigate any similar style of course, even if the specific obstacles changed. My main contribution was implementing a computer vision system that allowed us to do exactly that. A Raspberry Pi on the robot continuously captured images of the course ahead using a fisheye camera, and these photos were processed using a Python implementation of the extremely fast YOLO network, running on the Pi. The raspberry pi would then calculate how much the robot needed to turn in order to face the Ewok, and then pass this information to our main STM32 microcontroller, which controlled the motors.

Ball Bouncing in a Ball Motion Simulation

Personal project (Oct 2018)

Recently, I've been really interested in the field of computational physics. In order to get some experience solving computational problems, I decided to implement the simulation outlined in this paper by Tanner Morrison. The code was written in MATLAB and can be found on my github.

The code performs finite difference analysis to simulate the motion of both balls, and the result of any impacts that occur between them or the container around them. The weight of each ball and the initial position and velocity can all be set to vary the outcome. In the future, I hope to add in friction and rolling, to make the model more accurate.

UBC Solar

Engineering student design team (Sept 2016 - Sept 2017)

During my first year at UBC, I was a member of the mechanical design team for UBC Solar, an engineering student team building a solar powered race-car for the Formula Sun Grand Prix.

While working with UBC Solar, I had the opportunity to work independently and in groups to model components in AutoCAD and machine them using a water-jet cutter and a mill, perform finite element analysis using Ansys to evaluate the car's safety, and design the car's battery safety box.
