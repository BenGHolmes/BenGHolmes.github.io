  • UBC Solar

    September 2016 - September 2017

    ME

    I spent a year as a mechanical engineering team member on UBC Solar, a student design team building and racing 100% solar powered cars. While on the team, I design mounting brackets for the front suspension and steering rack using SolidWorks, and fabricated them with a waterjet cutter. I also performed finite element analysis for the rear suspension using Ansys, to ensure the car would survive a 1G turn and a 2G bump, and wrote a significant portion of the mechanical design report justifying our design decisions. My last project with the team was to work in a multidisciplinary team to design the car's battery safety box, which had a specific mechanical, electrical and ventilation requirements.

  • Autonomous Robot

    June - August 2018

    ML
    SW
    ME
    GitHub Repo Project Website

    First machine learning project! As part of a project course, we were tasked with building an autonomous robot capable of navigating a complex course and retrieving stuffed animals along the way. My team and I decided to take a risk for the sake of learning something cool, and decided to use computer vision to detect the stuffed animals, which had never been attempted before in that course.

    I spearheaded the implementation of the computer vision system. After tesing several of our own networks in Keras, we deciced to use the extremely fast YOLO (you only look once) network since it was the only one that could run on a Raspberry Pi at a decent frame rate. I then trained the network on ~2500 images, and wrote the Python code that took the network's predictions and converted them to a heading for our robot to follow.

    Since this was a large, multidisciplinary problem, I also had a hand in the mechanical design and fabrication, writing a PID algorithm for the motor controls, and designing an 8 bit digital to analog converter

  • Physics Simulation

    October 2018

    SW
    GitHub Repo

    Recently, I've been really interested in the field of computational physics. In order to get some experience solving computational problems, I decided to implement the simulation outlined in this paper by Tanner Morrison. The code was written in MATLAB and can be found on my github.

    The code performs finite difference analysis to simulate the motion of both balls, and the result of any impacts that occur between them or the container around them. The weight of each ball and the initial position and velocity can all be set to vary the outcome. In the future, I hope to add in friction and rolling to make the model more accurate.

  • WikiGame Bot

    November 2018

    DS
    SW
    GitHub Repo

    The WikiGame is a game where players race to find the fastest path between two random Wikipedia pages by following links in articles. To guarantee that I always beat my friends, I decided to build a bot to play using a graph of all connected Wikipedia pages, rather than guessing what the fastest path is myself.

    I downloaded the latest Wikipedia XML dump (a 50GB xml file), pulled all the links from every Wikipedia page, and stored them in a SQL database for quick access during games. I then used Selenium to interact with the game itself and follow the links on the pages. When a game starts, the bot grabs the start and end pages, and does a breadth first search of the graph to find the 20 fastest paths between the two pages (you get extra points for repeated wins within the time limit). I then quickly follow the links, completing as many paths as possible within the two minute time limit. The bot regularly scores 5-10x higher than its human opponents.
